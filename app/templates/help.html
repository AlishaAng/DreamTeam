Kinase Search

Kinase search allows you to type your query kinase into the search bar.

This returns all kinase entries matching your query, taking into account gene name and aliases. If kinase name is unknown, you can search for part of the name or a single letter to return all entries featuring your query. From the table of results click on choosen kinase gene name to view more information.

Kinase Results

Firstly background information about the target kinase is displayed. Using tabs on the results table you can explore phosphosites the kinases acts on as well as the (Uniprot) protein feature viewer for the target kinase.
